# API Perplexity

## Installation

```powershell
pip install -r requirements.txt
```

Le package PyPI a utiliser est `pplx` (pas `perplexity`).

## Avoir clé API pour crédit API (Payant - Très chère)

puis:

Sous PowerShell (Windows)
powershell
$env:PERPLEXITY_API_KEY="PPLX-xxxxxxxxxxxxxxxxxxxxxxxx"
Sous CMD
cmd
set PERPLEXITY_API_KEY=PPLX-xxxxxxxxxxxxxxxxxxxxxxxx
Sous Linux / macOS
bash
export PERPLEXITY_API_KEY="PPLX-xxxxxxxxxxxxxxxxxxxxxxxx"
